# Informatic learn - Платформа для изучения информатики и подготовки к ЕГЭ

## Описание
Веб-платформа для управления пользователями и банком заданий ЕГЭ по информатике с поддержкой бесконечной прокрутки и динамической фильтрации.

## Основные функции

### Управление пользователями
- **Администратор**: полный доступ ко всем пользователям и функциям
- **Учитель**: может добавлять и управлять только своими студентами
- **Студент**: доступ только к просмотру заданий

### Банк заданий
- 27 типов заданий ЕГЭ по информатике
- Подтипы для каждого типа задания
- Фильтрация по типу, подтипу и сложности
- Поиск по тексту задания
- **Бесконечная прокрутка** - задания подгружаются автоматически
- Поддержка HTML разметки и MathJax формул
- Загрузка изображений и файлов к заданиям

### Группировка студентов
- Создание групп студентов
- Назначение студентов в группы
- Управление группами

### Массовый импорт
- Импорт заданий из JSON файлов
- Автоматическое определение сложности
- Поддержка HTML разметки

## Технологии
- **Backend**: Django 5.2.4
- **Frontend**: Bootstrap 5, JavaScript (ES6+)
- **База данных**: SQLite (для разработки)
- **Математические формулы**: MathJax 3

## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/ВАШ_USERNAME/НАЗВАНИЕ_РЕПОЗИТОРИЯ.git
cd hack_ege_new
```

### 2. Создание виртуального окружения
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Применение миграций
```bash
python manage.py migrate
```

### 5. Создание администратора
```bash
python manage.py create_admin
```

### 6. Запуск сервера
```bash
python manage.py runserver
```

Откройте браузер и перейдите по адресу: http://127.0.0.1:8000/

## Использование

### Первый вход
1. Войдите как администратор (логин: admin, пароль: admin_psw)
2. Смените пароль в профиле
3. Добавьте учителей и студентов

### Добавление заданий
1. Перейдите в "Банк заданий"
2. Нажмите "Добавить задание"
3. Заполните форму и загрузите файлы при необходимости
4. Отметьте "HTML разметка" для заданий с формулами

### Массовый импорт
1. Подготовьте JSON файл с заданиями
2. Перейдите в "Массовый импорт"
3. Выберите тип заданий и загрузите файл

## Формат JSON для импорта
```json
[
  {
    "text": "<p>Текст задания с HTML разметкой</p>",
    "key": "правильный ответ",
    "difficulty": 0
  }
]
```

Где:
- `text` - текст задания (поддерживает HTML)
- `key` - правильный ответ
- `difficulty` - сложность (0=легкая, 1=средняя, 2=сложная)
