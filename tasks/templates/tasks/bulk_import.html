Формат JSON файла

JSON файл должен содержать массив объектов. Используются поля:

  • text - текст задания (HTML)
  • key - правильный ответ
  • difficulty - сложность (0=легкая, 1=средняя, 2=сложная)

Остальные поля игнорируются

Пример:
[
  {
    "text": "<p>Текст задания...</p>",
    "key": "ответ",
    "difficulty": 0
  }
]
